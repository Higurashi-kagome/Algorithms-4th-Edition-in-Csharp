# 1.3.6


## 解答

利用一个栈对队列元素进行反序操作。

先把队列中的元素全部入栈，再依次弹出并加入队列中。

## 代码

```csharp
var q = new Queue<string>();
q.Enqueue("first");
q.Enqueue("second");
q.Enqueue("third");
q.Enqueue("fourth");
var stack = new Stack<string>();
while (!q.IsEmpty())
    stack.Push(q.Dequeue());
while (!stack.IsEmpty())
    q.Enqueue(stack.Pop());

Console.WriteLine(q.ToString());
```
