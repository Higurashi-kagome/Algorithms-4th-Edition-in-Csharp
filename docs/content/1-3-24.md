# 1.3.24


## 解答

直接把该节点的 next 域设为 null，后续元素就会因无法访问而被清理掉。

## 代码

```csharp
var first = new Node<string>();
var second = new Node<string>();
var third = new Node<string>();
var fourth = new Node<string>();

first.Item = "first";
second.Item = "second";
third.Item = "third";
fourth.Item = "fourth";

first.Next = second;
second.Next = third;
third.Next = fourth;
fourth.Next = null;

var current = first;
while (current != null)
{
    Console.Write(current.Item + " ");
    current = current.Next;
}

RemoveAfter(second);
Console.WriteLine();

current = first;
while (current != null)
{
    Console.Write(current.Item + " ");
    current = current.Next;
}

static void RemoveAfter<TItem>(Node<TItem> i)
{
    i.Next = null;
}
```
