# 1.1.9


## 解答

有两种方法，要么直接调用库函数，要么用书中给出的代码转换。

## 代码

```csharp
var n = 4;

// 1.直接转换 Convert.ToString(int, int) 第一个为要转换的数，第二个为要转换的进制
Console.WriteLine($"{Convert.ToString(n, 2)}");

// 2.转换为二进制数
var s = "";
for (var i = n; i > 0; i /= 2)
{
    s = (i % 2) + s;
}

Console.WriteLine(s);
```
