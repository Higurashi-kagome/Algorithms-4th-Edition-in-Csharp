# 1.2.12


## 解答

这里使用蔡勒公式来推算星期。

参考：http://www.cnblogs.com/mq0036/p/3534314.html

## 代码

```csharp
/// <summary>
/// 计算当前日期是星期几，返回对应的星期名称。
/// </summary>
/// <returns></returns>
public string DayOfTheWeek()
{
    var d = Day;
    var m = Month;
    var y = Year;
        
    if (m < 3)
    {
        m += 12;
        y--;
    }

    // 使用蔡勒公式计算，参见 http://www.cnblogs.com/mq0036/p/3534314.html
    var w = (d + 2 * m + 3 * (m + 1) / 5 + y + y / 4 - y / 100 + y / 400) % 7;

    return DayOfWeek[w];
}
```
