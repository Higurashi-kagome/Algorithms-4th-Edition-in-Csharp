# 1.2.5


## 解答

`string` 类型中的 `Uppercase()` 以及 `Substring()` 都不会改变原有字符串，而是新建一个字符串并返回。

因此输出仍然为 Hello World。

## 代码

```csharp
var s = "Hello World";
s.ToUpper();
s.Substring(6, 5); // C# 中两个参数分别代表子串起始下标和长度
Console.WriteLine(s);
```
